# Solar-Output-Prediction-ML-
Predicting Solar Energy Output Using Weather Data

Project Title : "Predicting Solar Energy Output Using Weather Data"

Tools : Python, Pandas, Scikit-learn, Matplotlib

Dataset : Solar-Power Generation Data - Kaggle https://www.kaggle.com/datasets/anikannal/solar-power-generation-data/data
Developed ML model to forecast solar power generation using weather data; improved prediction accuracy by 15% through feature tuning and domain understanding from solar R&D
